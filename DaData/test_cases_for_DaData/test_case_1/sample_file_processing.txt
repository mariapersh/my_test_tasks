Тест-кейс № 1. Обработка файла-образца.
Предварительные шаги:
1. Зарегистрироваться на сайте ( см. тест-кейс «Регистрация» )
2. Пополнить баланс ( см. тест-кейс «Пополнение баланса» )
3. Подготовить excel-файл с данными из образца ( см. аттач «файл-образец.xlsx»)

Шаги:
1. Зайти на главную страницу портала  “Дадата”.
2. Нажать на кнопку «Войти» в правом верхнем углу.
3. Войти в систему ( почта test+2@gmail.com, пароль 1 ).
4. Нажать кнопку «Выбрать файл».
5. Выбрать файл, подготовленный ранее.
6. Проверить типы данных в столбцах.

7. Нажать на кнопку «Просмотреть результат».

8. Нажать на кнопку «Обработать файл».
