Тест-кейс № 2.Обработка файла с двумя адресами в одном поле


Предварительные шаги:

1. Зарегистрироваться на сайте ( см. тест-кейс «Регистрация» )
2. Пополнить баланс ( см. тест-кейс «Пополнение баланса» )
3. Подготовить файл формата xlsx с двумя адресами в одном поле (см аттач “ два адреса в одном поле.xlsx”)
Шаги:
1. Зайти на главную страницу ресурса “Дадата”.
2. Нажать на кнопку «Войти» в правом верхнем углу.
3. Войти в систему ( почта test+2@gmail.com, пароль 1 ).
4. Нажать кнопку «Выбрать файл».
5. Выбрать файл,подготовленный раньше.
6. Проверить типы данных в столбцах.
7. Нажать на кнопку «Просмотреть результат».
8. Нажать на кнопку «Обработать файл».

Ожидаемый результат:

Файл, подготовленный ранее («файл с двумя адресами в одном поле.xlsx»), загружен, система проверила данные и обработала данные с двумя адресами в  одном поле.
Введённые два адреса в одно поле были стандартизированы следующим образом:
- Исходный адрес -> Москва, Норильская улица, 17 кв 25 Спб,Белышева, 5/6, кв 683.После обработки системой :
1. г Москва,ул Норильская, д 17, кв 25;
2. г Санкт-Петербург,ул Белышева, д 5/6, кв 683.
-Исходный адрес -> Спб, ул Петрозаводская 8,Спб коллонтай 5/1,кв 1324. После обработки системой:
1. г Санкт-Петербург, ул Петрозаводская, д 8;
2. г Санкт-Петербург, ул Коллонтай, д 5/1, кв 1324.

Наглядный результат обработанных системой двух адресов в одном поле см в приложенном файле «два адреса в одном поле 2.xlsx».
